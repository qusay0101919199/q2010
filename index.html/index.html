🐞 أنواع البرمجيات الخبيثة الإضافية 🐞 Additional Types of Malware

تشمل أنواع البرمجيات الخبيثة المنتشرة والمرتبطة بسرقة البيانات أو تحقيق الربح ما يلي:

Common types of malware related to data theft or profit include:

  • Adware - برامج إعلانات مدفوعة أو منبثقة Adware - Paid or popup advertising software
  • Spyware - برامج تجسس تقوم بجمع المعلومات دون إذن المستخدم Spyware - Collects information without user consent
  • Botnets - شبكات روبوتية تستغل أجهزة المستخدمين عن بُعد Botnets - Networks that remotely control user devices
  • Keystroke logging - برامج تسجل ما يكتبه المستخدم بلوحة المفاتيح Keystroke logging - Records what the user types
  • Diallers - أدوات تتصل تلقائيًا بخطوط هاتفية ذات رسوم عالية Diallers - Automatically dial premium phone lines

👾 البرامج الخبيثة 👾 Malware

هي برامج ضارة تقوم بتحميل نفسها تلقائيًا دون أخذ الموافقة ❗ Malware is harmful software that installs itself automatically without user consent ❗

  • ( تتجاوز أمن النضام ) الأبواب الخلفية 🚪🕳️ Backdoors 🚪🕳️
  • ( تكون مخفية ) برمجيات خبيثة /span> Viruses attached to legitimate programs

🛑 برامج الفدية 🛑 Ransomware

برامج الفدية: هي برامج ضارة تقوم بتشفير ملفاتك أو تمنعك من استخدام جهازك حتى تدفع المال.

Ransomware: Malware that encrypts your files or blocks access to your system until a ransom is paid.

🛡️ الحماية من برامج الفدية: 🛡️ Protection from Ransomware:

  • تحديث البرامج ونظام التشغيل بانتظام. Regularly update software and OS.
  • عدم النقر على روابط مشبوهة. Avoid clicking suspicious links.
  • عدم فتح مرفقات بريد إلكتروني غير موثوق بها. Do not open untrusted email attachments.
  • تثبيت برامج الحماية وتحديثها. Install and update security software.
  • النسخ الاحتياطي المنتظم للبيانات. Regularly back up your data.
