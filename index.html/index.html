تحويل الشيء غير المفهوم إلى بيانات.

Converting the incomprehensible data back to its original form.

🖱️ ما هو Macro : 🖱️ What is a Macro:

شيء يُخزن في مجموعة، يمكن أن يكون ضارًا أو مفيدًا لإعادة الاستخدام 🔁

A stored item in a group that can be harmful or useful for reuse 🔁

⚙️ طرق التشغيل أو الإيقاف: ⚙️ How to Start or Stop:

  • 1. نذهب إلى Office 1. Go to Office
  • 2. ثم View 2. Then View
  • 3. ثم Macros Record 3. Then Macros Record
  • 4. ثم Name، نختار مثلاً font 4. Then Name, e.g., font
  • 5. نختار اللي بدنا إياه 🎨 5. Choose what you want 🎨
  • 6. ثم Stop Record ⏹️ 6. Then Stop Record ⏹️

📁 كيف تحط رمز على ملف: 📁 How to Add Password to a File:

  • 1. نفتح Office 1. Open Office
  • 2. ثم File 2. Then File
  • 3. ثم Save As 3. Then Save As
  • 4. ثم Browse 4. Then Browse
  • 5. نختار مثلاً Desktop 5. Choose e.g., Desktop
  • 6. ثم Tools 6. Then Tools
  • 7. ثم General Options 7. Then General Options
  • 8. بعدين نختار Password 🔒 8. Then Choose Password 🔒

✅ إيجابيات Password: ✅ Password Benefits:

  • 1. لا يمكن القراءة بدون كلمة المرور ❌📖 1. Cannot be read without the password ❌📖
  • 2. المستقبل المصرح به فقط يستطيع قراءة البيانات المشفرة 🔐 2. Only authorized receiver can read encrypted data 🔐

❌ قيود التشفير: ❌ Encryption Limitations:

  • 1. إذا ضاع التشفير لا يمكن القراءة 🔑🚫 1. If encryption is lost, data cannot be read 🔑🚫

🐞 أنواع البرمجيات الخبيثة

تشمل أنواع البرمجيات الخبيثة المنتشرة والمرتبطة بسرقة البيانات أو تحقيق الربح ما يلي:

  • Adware - برامج إعلانات مدفوعة أو منبثقة
  • Spyware - برامج تجسس تقوم بجمع المعلومات دون إذن المستخدم
  • Botnets - شبكات روبوتية تستغل أجهزة المستخدمين عن بُعد
  • Keystroke logging - برامج تسجل ما يكتبه المستخدم بلوحة المفاتيح
  • Diallers - أدوات تتصل تلقائيًا بخطوط هاتفية ذات رسوم عالية

🛑 برامج الفدية

برامج الفدية: هي عبارة عن برامج ضارة تقوم بتشفير ملفاتك أو تمنعك من استخدام جهاز الكمبيوتر الخاص بك حتى تقوم بدفع المال (الفدية) ليتم إلغاء قفلها.

🛡️ الحماية من برامج الفدية:

  • تحديث البرامج ونظام التشغيل بانتظام.
  • عدم النقر فوق الروابط المشبوهة (زيارات مواقع ويب غير موثوقة أو زائفة).
  • عدم فتح الروابط أو الرسائل غير الآمنة في البريد الإلكتروني أو وسائل التواصل الاجتماعي أو الرسائل النصية.
  • عدم فتح مرفقات بريد إلكتروني غير موثوق بها.
  • عدم التنزيل من مواقع ويب غير موثوق بها.
  • تجنب استخدام أجهزة USB غير معروفة.
  • تثبيت برامج الحماية وتحديثها.
  • النسخ الاحتياطي المنتظم للبيانات الخاصة بك.
